

<p>
MSE mede a diferença numérica entre valores previstos e reais (erro ao quadrado). É adequada para regressão, onde a saída é um valor contínuo (ex: prever preço de uma casa).
</p>

<p>
CrossEntropyLoss mede o quão errada é a distribuição de probabilidade prevista em relação à classe correta. É adequada para classificação, onde a saída é uma classe discreta (ex: identificar dígitos no MNIST).
</p>

<p>Em sintese: usamos MSE para regressão e CrossEntropy para classificação.</p>
